# Netflix Data Analysis

## Project Overview

This project analyzes Netflix content data using Python, Pandas, and Matplotlib. It helps understand content distribution, genres, and latest releases through data analysis and visualization.

## Features

* Movies vs TV Shows Analysis
* Most Common Genre Detection
* Latest Content Analysis
* Content Distribution Graph
* CSV Report Generation

## Technologies Used

* Python
* Pandas
* Matplotlib

## Dataset

The dataset contains:

* Title
* Type
* Genre
* Release Year

## Output

The project generates:

* Content analysis report
* Bar graph visualization
* `netflix_report.csv` file
